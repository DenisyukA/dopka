# Watermark Console Editor

## Опис

Консольний додаток для базової обробки зображень у C#:
- Відкриває та зберігає зображення (JPEG, PNG)
- Перетворює у відтінки сірого (ручна реалізація)
- Накладає водяний знак по всій площині
- Веде історію операцій (`history.json`)
- Меню — навігація стрілками (Up/Down/Enter), все інтуїтивно

---

## Архітектура та структура проєкту

Проект побудований на модульному принципі, де кожен клас виконує свою єдину відповідальність.

/WatermarkProject/
* Program.cs
* ImageEditor.cs
* **HistoryManager.cs**
* **FileValidator.cs**
* README.md
* history.json



- **Program.cs** — головний клас, меню, логіка, логування, обробка помилок
- **ImageEditor.cs** — вся робота із зображеннями (через SkiaSharp)
- **HistoryManager.cs** — **Узагальнений (Generic) клас**, відповідає за серіалізацію/десеріалізацію та керування історією дій у файл `history.json`.
- **FileValidator.cs** — статичний клас, що забезпечує валідацію вхідних файлів (перевірка існування та розширення).
- **history.json** — автоматично зберігається, містить історію дій

## Використані технології

- **SkiaSharp:** сучасна .NET-бібліотека для роботи з зображеннями
- **System.Text.Json:** для логування історії дій у форматі JSON
- **Власні структури:** (Option, OperationLog, Exception) — для чистоти коду та зручності
- **ООП:** весь код поділений на класи, немає дублювання

---

## Як запустити

1. Встановити .NET 6+ та SkiaSharp (NuGet)
2. Запустити через `dotnet run`
3. Меню керується стрілками, вибір — Enter, всі інструкції на екрані
